# WhatDaDev

Единый дом проекта: редизайн сайта WhatDaDev 2.0. Вся документация и контекст — здесь, чтобы не было путаницы.

## Структура папки

| Папка | Что внутри | Формат |
|-------|-----------|--------|
| [`docs/`](docs/README.md) | **База знаний / спека** — единый источник правды (бриф, карта сайта, контент-модель, дизайн, SEO, интеграции, инфра, решения ADR). Версионируется. | Markdown (канон) |
| [`deliverables/`](deliverables/README.md) | **Готовые экспорты для заказчика** — то, что отправляем на согласование. | .docx / .xlsx |
| [`references/`](references/README.md) | **Референсы и визуальный контекст** — разобранные примеры, скрины, мудборды. | Изображения |
| *(позже)* `src/` | Код сайта (фронт + CMS) — на этапе разработки. | — |

## Правило, чтобы не было путаницы
1 файл = 1 место. Markdown-документы живут только в `docs/`. Экспорты (.docx/.xlsx) — только в `deliverables/` (их пере-генерируем из `docs/`, не правим вручную). Картинки — только в `references/`.
