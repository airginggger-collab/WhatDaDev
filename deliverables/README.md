## Презентация для знакомства (v0.76)

5 слайдов: О нас · ELMA365 · Coordo · Наши проекты (предварительно) · Экосистема + контакты. Факты — из `docs/12-copy/` и `src/data/*.json`, метрики не выдуманы; слайд «Проекты» — заглушки под реальные кейсы/логотипы (Фаза 0).

**Идентичность дека осознанно отличается от сайта** (по запросу заказчика для этого демо): холодная стально-синяя палитра (`#134C7A`) + шрифт **IBM Plex Sans / IBM Plex Mono** вместо сайтовых тёплой бумаги + Piazzolla (ADR-0013/0014). Контраст всех пар проверен по WCAG AA. **Сайт это НЕ меняет.**

Пересборка PDF из исходника (шрифты встроены; правки — в `..._исходник.html`):
```bash
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless=new \
  --no-pdf-header-footer --print-to-pdf=out.pdf "file://<путь>/..._исходник.html"
```
PPTX собран отдельно через pptxgenjs (редактируемый; для точного вида поставить бесплатные шрифты IBM Plex).
