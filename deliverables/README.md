## Презентация для знакомства (v0.80 — продающая, 7 слайдов)

Продающий нарратив (по ресёрчу трендов B2B-питчей 2026), порядок: **О нас** (первый слайд, dogfooding) · **Проблема** (Before/After — крючок) · **ELMA365 — доверие** (крупная цифра 4000+) · **Coordo — как работает** (Месяцы→Недели + process-flow) · **Доказательство** (формат кейса, заглушки под цифры Фазы 0) · **Экосистема + импортозамещение** · **Следующий шаг** (CTA + контакты). Инфографика на каждом слайде. Факты ELMA — с elma365.com/ru; метрики WhatDaDev не выдуманы.

**Идентичность дека отдельная от сайта** (запрос заказчика для этого демо): **моно-синяя корпоративная** палитра — яркий электрик-азур `#1668C0` + глубокий navy `#123F73`, светло-синие тонированные плитки; оранжевого/жёлтого нет. Шрифты **IBM Plex Sans / IBM Plex Mono** (в PDF/HTML). Контраст — WCAG AA. **Сайт и ADR-0013/0014 это НЕ меняет.**

- **PDF/HTML** — на IBM Plex (шрифты встроены в base64 → точный вид на любой машине).
- **PPTX** — на **Arial + Courier New** (кириллица есть везде → без «квадратов» □□□; редактируемый). Не Plex, зато открывается корректно у всех. Прошлая версия на Plex давала □□□ без установленного шрифта.

Пересборка PDF из исходника (правки — в `..._исходник.html`):
```bash
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless=new \
  --no-pdf-header-footer --print-to-pdf=out.pdf "file://<путь>/..._исходник.html"
```
PPTX собран отдельно через pptxgenjs (скрипт — вне репо, в рабочей папке сессии).
