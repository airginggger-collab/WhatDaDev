# 09 — Контент-план (Фаза 1, черновик)

Статус: 🟡 черновик · версия v0.4
Привязка: утверждённая структура (`02-sitemap-ia`) × семантическое ядро (`05-seo`, 135 запросов).
Уточним очередность после реальных частот из Wordstat (Фаза 0, блок 3).

## Логика
Сначала — фундамент сайта и «быстрые победы» SEO (низкая конкуренция + COMM-интент), затем средняя игра (сравнения, отрасли), затем долгая (ВЧ-голова через блог).

## Волна 1 — фундамент + быстрые победы 🟢
Базовые страницы сайта + контурные/нишевые запросы, где конкуренция слабая.

| Страница | Цель / кластер | Тип |
|---|---|---|
| Главная `/` | Хаб 50/50, бренд | Каркас |
| `/products/express` + 3 контура | Согласование договоров, ОРД, корреспонденция (кл. 4, 5) | Продуктовый pillar |
| `/products/modules` + 5 модулей | Замещения, аудит СМК и др. (кл. 6) | Продукт |
| `/services` + `/services/vnedrenie` | «внедрение ELMA365 под ключ» (кл. 1) | Сервисный pillar |
| `/services/razrabotka` | Интеграции/доработка ELMA (кл. 9) | Услуга |
| `/elma365` | Что за платформа + dogfooding | Доверие |
| `/company/about`, `/contacts` | База доверия + конверсия | Базовые |
| KZ-локаль (ключевые страницы) | elma365 алматы/астана, СЭД КЗ (кл. 12) | SEO-гео |

## Волна 2 — средняя игра 🟡
Запросы выбора и отраслевые — высокая конверсия, конкуренты их не делают.

| Страница | Цель / кластер |
|---|---|
| Сравнительные лендинги | «коробка vs разработка», «ELMA365 vs Bitrix24/Directum» (кл. 10) |
| Отраслевые лендинги `/industries/*` | ЭДО/ELMA для производства, ритейла, девелопмента, логистики, финансов (кл. 11) |
| `/products/express/pricing`-блок | «стоимость внедрения», калькулятор (кл. 1, 3) |
| Остальные направления `/services/vnedrenie/*` | CRM, ЭДО, ServiceDesk, закупки, маркетинг, проекты, КЭДО (кл. 1, 3, 7, 8) |
| `/company/success-stories`, `/cases` | Анонимные кейсы с цифрами |

## Волна 3 — долгая игра + регулярка 🔴
Топикальный авторитет: ВЧ-голову берём блогом, а не лендингами.

| Контент | Цель / кластер |
|---|---|
| Блог `/press/articles` — кластеры статей | ЭДО/СЭД/BPM/КЭДО общие, «боль» (кл. 2, 3, 7, 8, 15) |
| `/press/glossary` | Глоссарий терминов (инфо-хвост, внутренняя перелинковка) |
| `/press/reviews` | Обзоры релизов ELMA (свежесть + экспертность) |
| Регулярные публикации | 1–2 статьи/мес под незакрытые запросы ядра |

## Принципы написания (для копирайтера/нейро)
- 1 страница = 1 кластер/интент; не дублировать (помним каннибализацию «услуга ЭДО» vs «продукт Экспресс внедрение»).
- На каждой коммерческой странице: H1 под запрос, цифры/доказательства, 1 CTA, FAQ (под schema FAQPage).
- Цены показываем (преимущество против конкурентов).
- Внутренняя перелинковка pillar ↔ кластерные страницы ↔ статьи.

## Зависимости
- Реальные частоты (Wordstat) → финальная очередность внутри волн.
- Кто пишет тексты (Фаза 0, блок 2) → объём нашей работы.
- Кейсы/цифры (Фаза 0, блок 2) → наполнение блоков доверия.
