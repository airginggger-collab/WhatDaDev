# 05 — SEO

Статус: 🟡 черновик · версия v0.1
Семантическое ядро: `../../deliverables/WhatDaDev_Семантическое_ядро.xlsx` (135 запросов, 15 кластеров, привязка к страницам).

## Принцип (ADR-0004)
Не воюем за ВЧ-«голову» («электронный документооборот», «СЭД», «BPM», «КЭДО») — там домены-гиганты. Берём **длинный хвост по формуле** `[согласование договоров / ОРД / маршруты / замещение / корреспонденция] × [коробочный / на базе ELMA365 / без кода / on-premise]` + сравнения + отрасли + Казахстан.

## Где Яндекс/Google
В РФ решает **Яндекс** (коммерческие + поведенческие факторы). В КЗ — Яндекс + Google. Оптимизируем под оба: Я.Вебмастер + Search Console.

## Приоритеты (из ядра)

## Технический SEO
- SSR/SSG (Astro/Next), идеальные Core Web Vitals, чистые ЧПУ, XML-sitemap, robots.
- Schema.org: Organization, SoftwareApplication+Offer (Coordo), Service, FAQPage, Article, BreadcrumbList, Review (когда будут).
- hreflang RU↔EN.
- **Карта 301** старый→новый URL — `redirects.md` (TODO, нужен доступ к текущим позициям).
