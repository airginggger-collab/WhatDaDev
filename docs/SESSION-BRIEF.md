## Кто мы и что делаем

**WhatDaDev** — компания по внедрению ELMA365 (российская BPM/СЭД-платформа). Собственный продукт — **«Экспресс внедрение (Coordo)»**: пакетное внедрение за 45 дней для SMB.

Сайт — **B2B-лидген**: посетитель → заявка/демо → менеджер → ELMA365 CRM → сделка. Никакого e-commerce, никаких онлайн-платежей.

Живой сайт: **https://whatdadev.airg-inggger.workers.dev/**
Целевой домен: https://whatdadev.ru (пока WordPress, привязка — отдельная задача)

## Стек и деплой

- **Astro 5 (SSG)** — 68 страниц статически генерируются в `dist/`
- **Cloudflare Workers Static Assets** — `wrangler.jsonc`, автодеплой при `git push main`
- **Стили** — обычный CSS + CSS-переменные в `src/styles/global.css` (без Tailwind)
- **Токены** — `docs/04-design/tokens.json`, акцент `#1F5B99` (корпоративный синий, ADR-0014), шрифты Piazzolla + Golos Text + IBM Plex Mono (ADR-0013)
- **Контент** — в `src/data/*.json` и markdown-коллекциях `src/content/`; редактируется через кабинет Sveltia на `/admin/` (ADR-0015). Разметка страниц — в `.astro`.
- **Коммит-автор обязателен**: `git -c user.email=airg.inggger@gmail.com -c user.name=airginggger-collab commit -m "..."`
- **После каждого коммита — push**: `git push origin main`

Команды:
```bash
npm run dev      # → http://localhost:4321
npm run build    # → dist/
```

## Правила агента (из CLAUDE.md)

1. **Язык** — всегда отвечать и писать доки на **русском**
2. **Факты из кода** — URL, имена файлов, команды только из реального `src/`
3. **Авто-цикл доки→commit→push** после каждого значимого изменения (правило 2026-07-06; отменило «не коммитить без команды»)
4. **Доки синхронно** — каждое изменение → `docs/CHANGELOG.md` + `docs/HANDOFF.md`
5. **Без эмодзи** в `.astro`-файлах и контенте (B2B-тон)
6. **Хлебные крошки** — обязательны на каждой странице кроме `/` и 404
7. **Не добавлять Cloudflare-адаптер** — сайт статический, адаптер ломает деплой
8. **Учиться на поломках** — что-то сломалось → разобрать что/почему и записать в `docs/LESSONS.md`, чтобы не повторять (правило 2026-07-10)
