# 03 — Модель контента (для CMS)

Статус: 🟡 черновик · версия v0.1
Продукт CMS — 🔴 отложен (ADR-0006). Кандидаты: self-hosted Directus / PayloadCMS.
Требование зафиксировано: **менеджер обновляет контент без кода**.

## Требования к CMS
- Дружелюбная админка для не-технического менеджера.
- Локализация **RU / EN** на уровне полей.
- Состояния `draft → review → published` + **история ревизий** (откат).
- Preview черновика до публикации.
- Роли: `admin` (разрабы — схема/код), `editor` (менеджер — только контент).
- Self-hosted (данные РФ/КЗ, без US-резидентности) — предпочтительно.

## Коллекции

| Коллекция | Ключевые поля |
|---|---|
| `pages` | slug, заголовки/блоки (локализ.), seo_meta |
| `coordo_modules` | название, описание, иконка, скриншот, порядок |
| `services` | название, описание, «цена от», иконка, slug |
| `cases` | отрасль, размер компании, метрики[], текст, slug, logo (скрыт до согласия), отрасль-тег |
| `testimonials` | текст, автор-должность, компания (опц.), фото (опц.) |
| `industries` | отрасль, заголовок, контент, seo_meta, slug |
| `blog_posts` | заголовок, контент, автор, дата, теги, обложка, seo_meta, slug |
| `team` | имя, роль, фото, порядок |
| `settings` (singleton) | контакты, реквизиты РФ/КЗ, соцсети, баннер акции (вкл/выкл + текст + дата) |
| `seo_meta` (встраив.) | title, description, og_image, schema_type, noindex |
| `redirects` | from, to, type (301/302) — менеджер ведёт карту переездов |
| `navigation` | пункты меню (header/footer), порядок, ссылки |

## Связи
- `cases.отрасль-тег` → `industries` (перелинковка для SEO).
- `blog_posts.теги` → кластеры семантического ядра (см. `05-seo`).
- `coordo_modules` → страницы `/coordo/*`.
