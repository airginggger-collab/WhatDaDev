# 10 — Бюджет

Статус: 🟢 направление утверждено · версия v0.5
Решения: без e-commerce (ADR-0008); разработка «вы + я», старт — максимум бесплатно (ADR-0009).

## Вне scope (0₽)
Продажа через сайт, корзина, оформление заказов, онлайн-оплата, Kaspi/эквайринг, PCI. Сайт = B2B-лидген: заявка/демо → ELMA CRM → счёт/договор офлайн.

## Старт — бесплатно (~0₽/мес)
| Что | Решение | Цена |
|---|---|---|
| Хостинг (SSG) | Cloudflare Workers Static Assets (free tier) | 0₽ |
| Домен whatdadev.ru | уже есть | ~200–500₽/год |
| SSL | Let's Encrypt / CF | 0₽ |
| Аналитика | Я.Метрика + Вебмастер + GA/GSC | 0₽ |
| Семантика | Я.Wordstat | 0₽ |
| Формы → лиды | ELMA CRM + Telegram-уведомление | 0₽ |
| CMS (старт) | контент в Git (MD/MDX), правим сами | 0₽ |
| CMS (когда менеджеру самому) | Decap/TinaCMS (Git-based) | 0₽ |
| Шрифты / дизайн | Google Fonts / Figma free | 0₽ |

Разработка: вы + я → дев-расходы 0₽.

## Подключаем по триггеру (платно)
| Триггер | Решение | Ориентир РФ 2026 |
|---|---|---|
| РФ-резидентность / скорость в Яндексе | RF-VPS (Timeweb/Selectel) | ~500–1 500 ₽/мес |
| Богатая админка для менеджера | Directus/Payload на VPS | входит в VPS |
| Глубокая семантика | Keys.so | ~1 500–4 000 ₽/мес |
| Рассылки | Unisender/Sendsay | free → ~1 500 ₽/мес |
| Чат / запись на демо | Jivo / Calendly | free → ~990 ₽/мес |
| Мониторинг | Sentry / UptimeRobot | free |

## Caveat
Free-хостинги — глобальные/US. Если РФ-резидентность станет юр-требованием → RF-VPS (~500₽/мес). На старте free допустимо.
