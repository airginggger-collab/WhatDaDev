# ADR-0013 — Уход от «ИИшного» дизайна: инженерно-издательская идентичность (Piazzolla + Golos + зелёный)

Статус: ✅ Accepted · 2026-07-04 · Заменяет визуальную часть ADR-0011

## Контекст
Пользователь зафиксировал задачу сессии: «важно избавиться от ИИшного дизайна». Аудит подтвердил: идентичность из ADR-0011 собрана из дефолтов, которые генерируют все ИИ-инструменты — шрифт Inter, акцент #4F46E5 (стоковый Tailwind indigo-600), вся палитра — стоковые slate-оттенки Tailwind, тёмный hero с фиолетовым свечением и сеткой, mac-окно с тремя точками, bento-грид с фейковыми дашбордами, pill-кнопки 9999px, hover-lift карточек. Сайт выглядел как шаблонный ИИ-лендинг, а не живой продукт.

## Решение
Направление — **инженерно-издательское** (серьёзный B2B-тон, «живая инженерная компания»):

- **Шрифты (self-hosted через Fontsource, без Google CDN):**
  - Заголовки H1/H2 — **Piazzolla Variable** (сериф, полноценная кириллица);
  - Текст/UI — **Golos Text Variable** (российский гротеск, кириллица-native);
  - Цифры, подписи, eyebrow, крошки — **IBM Plex Mono** (400/500).
- **Палитра — тёплая, не Tailwind:** бумага `#FAF8F2` / чернила `#191D1B` / тёплые границы `#E2DDCE`; тёмные секции — чернильно-зелёные (`#101714`/`#18211C`); **акцент — глубокий зелёный `#0F6B4F`** (600 `#0B563F`, 50 `#EAF2EC`). Красного нет (ADR-0003 соблюдён), тёмный hero → светлый контент сохранён (ADR-0002).
- **Формы:** радиусы 16px → 6/4px, pill-элементы → прямоугольные; hover карточек — смена бордера + жёсткая тень `3px 3px 0` вместо lift+blur.
- **Снос клише:** сетка-фон и радиальное свечение hero, mac-точки у видео (осталась моно-подпись «запись экрана»), bento с фейковыми анимированными интерфейсами (заменены реальными скриншотами ELMA365 из `public/covers/` с подписями), кольцо-метрика на главной (заменено стат-таблицей), градиентные дивайдеры.
- **Reveal — прогрессивное улучшение:** скрытие `.reveal` действует только при `html.js` (класс ставится инлайн-скриптом в `<head>`); hero-элементы анимируются чистым CSS (`fadeUp`), не ждут JS. Без JS весь контент виден (SEO/LCP).

## Почему
«Inter + indigo + slate + bento + glow» — самый узнаваемый маркер ИИ-генерации в 2024–26; для B2B-лидгена он подрывает доверие («сайт сделан за час»). Сериф в заголовках + моно в данных + тёплая бумага + один сдержанный зелёный — редкое в ИИ-шаблонах сочетание, при этом консервативное и уместное для корпоративного ЭДО. Реальные скриншоты продукта честнее фейковых анимаций (и уже были в репо). Self-host шрифтов надёжнее Google CDN для аудитории РФ.

## Последствия
- `global.css` переписан (API классов сохранён), `tokens.json` — v2.
- Инлайн-хардкоды индиго/slate заменены по всем `src/pages/*` (sed-свип).
- Демо-цифры кейсов остаются заглушками (Фаза 0) — теперь показаны один раз, в стат-таблице.
- `public/og.png` остался в старой айдентике — перегенерировать (бэклог).
- Дальнейшие правки цвета/шрифта — только новым ADR.
