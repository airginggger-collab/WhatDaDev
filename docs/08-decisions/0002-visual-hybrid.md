# ADR-0002 — Визуал: тёмный hero → светлый контент

Статус: ✅ Accepted · 2026-06-18

## Контекст
Заказчик любит премиум-минимализм (Apple). Разобранный Behance-концепт — тёмный+сервисный. Нужно отличаться от всех интеграторов ELMA.

## Решение
Гибрид: кинематографичный тёмный hero → светлый воздушный доказательный контент (паттерн Stripe/Vercel/Linear).

## Почему
Сочетает «премиум-серьёзность» (тёмный hero) и доказательную плотность B2B (светлый контент). Ни один интегратор ELMA так не выглядит — незанятая ниша. Выбор заказчика.

## Последствия
Перф-риск тёмного hero → SSR текста + отложенная анимация (см. 07-infra).
