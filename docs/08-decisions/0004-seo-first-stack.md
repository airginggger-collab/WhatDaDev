# ADR-0004 — SEO-first + SSR/SSG стек

Статус: ✅ Accepted · 2026-06-18

## Контекст
Заказчик дважды подчеркнул: находимость в поиске лучше конкурентов — ведущее требование.

## Решение
SEO в фундаменте: SSR/SSG (Astro/Next), schema.org, топикальные кластеры, hreflang RU/EN, карта 301. Не воюем за ВЧ-голову — берём хвост + сравнения + отрасли + КЗ.

## Почему
Конкуренты сильны брендом/доменом, но слабы технически и по контенту (прячут цены, нет сравнений/отраслевых). Атакуем их пробелы. В РФ решает Яндекс (коммерч.+поведенч. факторы).

## Последствия
Стек не может быть тяжёлым клиентским SPA. Тёмный hero не ломает Core Web Vitals. Нужны доступы к Wordstat/Вебмастеру для точных частотностей.
