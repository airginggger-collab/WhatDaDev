# ADR-0010 — Нейминг продукта: «Экспресс внедрение (Coordo)»

Статус: ✅ Accepted · 2026-06-18

## Решение
Гибрид: в заголовках/SEO ведём описанием («Экспресс внедрение ЭДО на ELMA365»), имя бренда **Coordo** используем как название продукта рядом — «Экспресс внедрение (Coordo)».

## Почему
Описание даёт SEO-трафик и ясность (никто не ищет «Coordo»), а имя строит бренд-актив под продукт с собственной дорожной картой до 2027. Лучшее из двух подходов. Решение заказчика.

## Последствия
- H1/Title — описательные; «Coordo» — подзаголовком/рядом, в продуктовых блоках.
- Фирстиль (04-design) учитывает суб-бренд Coordo внутри WhatDaDev.
