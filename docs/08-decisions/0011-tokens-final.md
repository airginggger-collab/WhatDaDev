# ADR-0011 — Финал токенов: светлый премиум (indigo + Inter)

Статус: ✅ Accepted · 2026-06-19

## Контекст
Заказчик прислал референсы (CAST AI, searchable, AgentQL) — светлый премиум с анимацией продукта. Нужно зафиксировать акцент и шрифт (были предварительными, ADR-0003).

## Решение
- Направление: **светлый премиум**, тёмный hero сохраняем (ADR-0002).
- **Акцент: electric-indigo** (#4F46E5 / 600 #4338CA / 50 #EEF0FE). teal — документированная альтернатива, в проде не используется.
- **Шрифт: Inter** (кириллица, чистый премиум-гротеск).
- Премиум-полировка: мягкий hover карточек, анимированные метрики/графики, reveal-on-scroll.

## Почему
Indigo сохраняет доверие (не тревожный красный) и читается премиально; Inter — чистый, поддерживает кириллицу, совпадает с эстетикой референсов. Уточняет, не отменяет ADR-0002/0003.

## Последствия
`tokens.json` — финал (v1). Дальнейшие правки цвета/шрифта — только новым ADR.
