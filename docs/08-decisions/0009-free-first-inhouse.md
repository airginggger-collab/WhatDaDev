# ADR-0009 — Free-first, разработка своими силами

Статус: ✅ Accepted · 2026-06-18

## Решение
Разработку ведут «заказчик + AI-ассистент». Старт инфраструктуры — максимум на бесплатных тарифах; платное подключаем только по триггеру (см. 10-budget).

## Почему
WhatDaDev — dev-студия (нет смысла платить подрядчику). Free-тарифы покрывают лидген-сайт. Запуск ≈ 0₽.

## Последствия
- CMS на старте — контент в Git (free); Git-based CMS (Decap/Tina) когда менеджеру нужно самому; Directus/VPS — позже по триггеру. Дополняет ADR-0006.
- Free-хостинг = US/глобальный; РФ-резидентность → RF-VPS при необходимости.
