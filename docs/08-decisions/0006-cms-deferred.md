# ADR-0006 — Выбор CMS отложен

Статус: ⛔ Superseded ADR-0015 (2026-07-05) · изначально 🟡 Deferred · 2026-06-18

> Снято: CMS выбрана и внедрена — Sveltia (git-based), см. ADR-0015 и `docs/19-cms-migration.md`.

## Контекст
Требование: менеджер обновляет контент без кода. Заказчик решил выбор продукта отложить.

## Решение
Фиксируем ТРЕБОВАНИЯ к CMS (см. 03-content-model), продукт выбираем на этапе стека. Кандидаты: self-hosted Directus (дружелюбная админка, ревизии) / PayloadCMS (тесная связка с Next).

## Почему
SaaS (Sanity/Contentful) — US-хостинг, риск резидентности/доступа для РФ. Self-hosted предпочтительно. Решение требует выбора стека, который пока открыт.

## Последствия
Content-model проектируется CMS-агностично, чтобы лечь на любой из кандидатов.
