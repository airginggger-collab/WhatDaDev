# ADR-0008 — Без e-commerce (только лидген)

Статус: ✅ Accepted · 2026-06-18

## Решение
Убираем продажу через сайт, корзину, оформление заказов, онлайн-оплату (включая Kaspi/эквайринг). Сайт — B2B-лидген: заявка/демо → ELMA CRM → счёт/договор офлайн.

## Почему
Продукты B2B (Coordo от 690k₽, услуги) продаются через счёт/договор, не по карте. E-commerce добавил бы шлюзы, эквайринг, юр-обвязку, PCI — стоимость и сложность без выгоды. Решение заказчика.

## Последствия
Платёжные интеграции = 0₽. Конверсия — заявки/демо, не заказы.
