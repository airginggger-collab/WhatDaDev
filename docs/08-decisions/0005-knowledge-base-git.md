# ADR-0005 — База знаний в Git /docs

Статус: ✅ Accepted · 2026-06-18

## Контекст
Нужна версионность спеки и сбор контекста для передачи бэк/фронт-разрабам.

## Решение
Единый источник правды — Git-репозиторий `/docs` (Markdown). Семантические версии + CHANGELOG + ADR. Документы для заказчика (.docx/.xlsx) — экспорт отсюда. Notion-зеркало решено НЕ делать (только Git).

## Почему
Разрабы живут в Git, бесплатная версионность и история. Один источник без рассинхрона.

## Последствия
Заказчику менее наглядно, чем Notion — компенсируем экспортом .docx на согласование.
