# ADR-0014 — Акцент: корпоративный синий #1F5B99 (вместо зелёного)

Статус: ✅ Accepted · 2026-07-05 · Уточняет ADR-0013 (идентичность сохраняется, меняется только акцентная ветка)

## Контекст
Пользователь отклонил зелёный акцент ADR-0013 (пробовали хвойный #0F6B4F и изумрудный #0D8148 — «зелёный оттенок не нра»). По правилу №6 (CLAUDE.md) предложены 4 варианта: петроль, корпоративный синий, охра, монохром. Пользователь выбрал **Б — корпоративный синий**.

## Решение
- **Акцент: #1F5B99** (hover #174873, 400 #3273B4, светлые 50 #E7EEF6 / 100 #D9E6F3 / 200 #BFD7EC / 300 #8FBBE0). Это глубокий корпоративный синий — НЕ индиго #4F46E5 (ИИшный дефолт, запрещён ADR-0013).
- **Тёмные фоны перетонированы** из зеленоватого в холодный сине-чернильный: dark-bg #10151A, dark-surface #171E26, dark-border #263140, dark-muted #99A3AE.
- Остальная идентичность ADR-0013 без изменений: Piazzolla + Golos + Plex Mono, тёплая бумага #FAF8F2, нейтральные подложки #DFE0DA / границы #C4C5BB (утверждены отдельно), радиусы 6/4px.

## Почему
Синий — самый конверсионный и доверительный акцент для консервативного B2B (ЭДО, документооборот); глубокий оттенок и серифная типографика удерживают «неИИшность». Контраст с белым на кнопках ~7:1 (AA+).

## Последствия
`tokens.json` v3 (accent + dark). Дальнейшие правки акцента — только новым ADR после утверждения вариантов пользователем.
