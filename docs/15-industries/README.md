# 15 — Отраслевые лендинги (Волна 2)

Статус: 🟡 черновик · версия v0.13 · интент COMM

## Шаблон отраслевой страницы
1. **Hero (тёмный):** H1 «ELMA365/ЭДО для <отрасли>» + подзаголовок (боль отрасли) + CTA.
2. **Боли отрасли** — 3–4 конкретные проблемы именно этой сферы.
3. **Что автоматизируем** — релевантные процессы (договоры, ОРД, закупки, согласование…).
4. **Почему ELMA365 здесь** — 2–3 аргумента под отрасль.
5. **Кейс/доверие** — {{метрика по отрасли}}, ссылка на /cases.
6. **FAQ** (под schema) + CTA.
Title: «<Что> для <отрасли> на ELMA365 | WhatDaDev».
