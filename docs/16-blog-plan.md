Движок топикального авторитета (Волна 3, `09-content-plan`). Темы — под инфо-кластеры ядра (`05-seo`).
URL: статьи — `/press/articles/<slug>`, глоссарий — `/press/glossary/<term>`. Интент в основном INFO (прогрев), ведём на коммерческие страницы перелинковкой.

## Принципы
- 1 статья = 1 запрос-кластер; в конце — перелинковка на профильную коммерческую страницу + CTA.
- ВЧ-«голову» (ЭДО/СЭД/BPM/КЭДО) берём именно статьями (не лендингами) — через пользу и полноту.
- Регулярность важнее объёма: {{1–2}} статьи/мес под незакрытые запросы.

## Правила SEO для статей
- H1 = запрос; структура H2/H3 под под-вопросы; FAQ-блок (schema FAQPage).
- Внутренние ссылки на pillar/коммерческие страницы (2–4 на статью).
- Дата публикации/обновления; автор-эксперт (E-E-A-T).
- Без воды; польза + примеры из наших внедрений {{по согласию}}.

## Зависимости
- Реальные частоты (Wordstat) → приоритет тем внутри волн.
- Кто пишет (Фаза 0, блок 2) → темп публикаций.
- Факты для сравнений (vs Directum/1С) — `{{verify}}` перед публикацией.
