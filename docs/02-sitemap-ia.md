## Важные замечания к структуре

- **Навигация ≠ полный список страниц.** Для SEO дополнительно создаём посадочные/контентные страницы ВНЕ основного меню (см. `05-seo`): отраслевые лендинги, сравнительные страницы («коробка vs разработка», «ELMA365 vs Bitrix24»). Живут как статьи в Пресс-центре или отдельные landing-URL.
- **Риск каннибализации:** `1.1.2 ЭДО (услуга внедрения)` и `2.1 Экспресс внедрение (продукт-ЭДО)` целятся в близкие запросы. Разводим интент: услуга = «внедрение ЭДО под задачи» (проектно), продукт = «готовый ЭДО из коробки, быстро». Разные H1/мета.
- **Замещения** переехали из «контуров» в `2.2.2 Модули и решения` — это отдельный продаваемый модуль, не часть Экспресс-внедрения.
- 🔴 Уточнить: бренд «Coordo» используется где-либо (на страницах продукта) или полностью заменён на «Экспресс внедрение»?

## Блоки ключевых страниц (вектор: тёмный hero → светлый контент)

### Главная — хаб 50/50
Тёмный hero (заголовок + «12 внедрений» + 2 CTA: Услуги / Продукты) → развилка 2 карточки → «почему мы» + dogfooding → анонимные кейсы с цифрами → доверие/сертификаты/гео → тёмный финальный CTA.

### /products/express (продуктовый pillar)
Тёмный hero «ЭДО из коробки, без полугода разработки» + интерфейс/схема маршрута + CTA «Записаться на демо» → проблема → 3 направления (договоры/ОРД/корреспонденция) → ядро маршрутов без кода → цена «разово, без подписки» + акция → безопасность/юр.значимость → демо-CTA.

### /services/vnedrenie (сервисный pillar)
Тёмный hero (результат + «12 внедрений») → 8 направлений (CRM…Прочие процессы) → этапы (степпер) → кейсы по отраслям с метриками → команда/сертификаты → CTA «Обсудить проект».
