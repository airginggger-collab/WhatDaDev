Статус: 🟡 черновик · версия v0.1

Принцип: **главная = хаб 50/50** → направляет в воронку и снимает доверие. Деньги в SEO приносят глубокие посадочные. Доверие и контент — общие.

## Блоки ключевых страниц

### Главная (тёмный hero → светлый контент)
1. Тёмный hero: заголовок-клин + позиционирование + «12 внедрений» + 2 равных CTA.
2. Развилка: 2 карточки (Coordo · Услуги).
3. Светлая: «почему мы» + dogfooding.
4. Анонимные кейсы с цифрами (3 главных).
5. Доверие: сертификаты, география РФ+КЗ.
6. Тёмный финальный CTA-якорь.

### /coordo (продуктовый pillar)
1. Тёмный hero: «ЭДО на ELMA365 — из коробки, без полугода разработки» + интерфейс/схема маршрута + CTA «Записаться на демо».
2. Проблема (долго/дорого/риск кастома).
3. 4 контура — именованные карточки → свои страницы.
4. Ядро маршрутов «без кода» (скриншот-демо).
5. Цена-преимущество: «разово, без подписки» + акция.
6. Безопасность / юр.значимость (подпись, аудит-трейл, Диадок/СБИС из roadmap).
7. Дорожная карта. 8. Демо-CTA.

### /services/vnedrenie-elma365 (сервисный pillar)
1. Тёмный hero: результат + «12 внедрений».
2. Экспертиза (CRM / Service Desk / КЭДО / BPM).
3. Этапы (горизонтальный степпер).
4. Кейсы по отраслям с метриками.
5. Команда / сертификаты. 6. CTA «Обсудить проект».
