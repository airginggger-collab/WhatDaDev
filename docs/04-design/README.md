# 04 — Дизайн

Статус: ✅ финал (ADR-0013/0014) · токены v3

## Направление (ADR-0002, ADR-0003)
**Гибрид: тёмный кинематографичный hero → светлый воздушный доказательный контент.**
- Тёмный hero: navy/графит + лёгкое свечение, крупный гротеск, продукт/схема как герой.
- Светлый контент: воздух, реальные скриншоты, цифры, кейсы.
- **Акцент НЕ красный** (красный = тревога/ошибка, против доверия в ЭДО).

## Финал идентичности (ADR-0013/0014)
- Акцент: **корпоративный синий `#1F5B99`** (hover `#174873`). НЕ красный (ADR-0003), НЕ indigo. Зелёные варианты отклонены пользователем.
- Шрифты: **Piazzolla** (H1/H2) + **Golos Text** (текст) + **IBM Plex Mono** (цифры/подписи), self-hosted через Fontsource.
- Тёплая бумага `#FAF8F2` + чернила; радиусы 6/4px, без pill/glow/bento.

## Токены (заполнить на этапе UI-kit)
Файл `tokens.json` — ✅ v3 (ADR-0014): палитра light/dark, акцент синий `#1F5B99`, шрифты Piazzolla/Golos/Plex Mono, типошкала, радиусы 6/4px, motion. Это источник токенов для `src/styles/global.css`.

## Визуальные шпаргалки (HTML, для Figma / Stitch)
Собраны из `tokens.json` + `src/styles/global.css` — открывать в браузере, значения переносить в дизайн-инструмент. Шрифты подключены относительными путями к `node_modules` (после `npm install` рендерятся в настоящих гарнитурах; иначе — фолбэки Georgia/system).
- **`tokens-cheatsheet.html`** — полная шпаргалка: типографика (гарнитуры, шкала desktop/mobile, веса, line-height), все цвета light/dark/accent/semantic с превью и hex, отступы, радиусы, тень, сетка, motion, токены компонентов. Для переноса в **Figma**.
- **`stitch-dark.html`** — тёмная тема под **Google Stitch**: готовый промпт EN/RU (копируется), маппинг «поле Theme → значение» с подписями EN/RU, все тёмные токены. Вставлять промпт в начало запроса Stitch.

## Паттерны показа продукта
- **Product-in-motion (референс: cursor.com).** Продукт показываем не статичными скриншотами, а **короткими зацикленными видео/гифками интерфейса в действии** (маршрут согласования проходит этапы, договор двигается по статусам, настройка маршрута мышкой). Бесшовный loop, muted, autoplay. Ложится на наш «интерфейс как герой» в тёмном hero и в блоках контуров.
- Где применяем: hero `/products/express`, карточки 3 контуров, блок «маршруты без кода».
- **Перф/доступность (обязательно, не ломать Core Web Vitals — ADR-0004):** формат webm/mp4 (не тяжёлый gif), `loading=lazy`, постер-кадр для LCP, автоплей только в зоне видимости (IntersectionObserver), fallback на статичный кадр при `prefers-reduced-motion`.
