# 01 — Бриф (стратегия и направление)

Статус: 🟡 черновик на согласование · версия v0.1
Канонический источник. `.docx` на Рабочем столе — экспорт для заказчика.

## Стратегия и приоритеты

| Параметр | Решение |
|---|---|
| Приоритет воронок | **50 / 50** — Coordo и услуги равны. Главная = хаб-распутье. (ADR-0001) |
| Дедлайн | Нет жёсткого. Качество и итерации. Акция Coordo — баннером. |
| Стоп-сегменты | Нет. Максимальный охват (усиливает SEO). |
| SEO | **Ведущее требование**: находимость выше конкурентов в Яндекс + Google. (ADR-0004) |
| Языки | RU (основной) + EN (зеркало ключевых страниц). EN-объём — 🔴 открыто. |
| CRM / заявки | ELMA365 (формы → собственная CRM, аргумент dogfooding). |

## Визуальное направление — гибрид (ADR-0002, ADR-0003)
Тёмный кинематографичный hero → светлый, воздушный, доказательный контент (паттерн Stripe/Vercel/Linear). Отличает от всех интеграторов ELMA, соответствует премиум-ориентиру (Apple).

| Слой | Решение |
|---|---|
| Hero (тёмный) | Navy/графит + лёгкое свечение, крупный заголовок, интерфейс Coordo / схема маршрута как герой, 1 CTA |
| Контент (светлый) | Светлый фон, воздух, крупный гротеск, реальные скриншоты, цифры, кейсы |
| Акцент | **НЕ красный.** Кандидаты: electric-indigo / teal. Финал — после Apple-референсов. 🔴 |
| Шрифт | Характерный гротеск (не дженерик). 🔴 |

## Доверие без имён клиентов
12 внедрений, 3 главных, имена пока нельзя. Строим доверие через: анонимные кейсы (отрасль + размер + метрика), число «12 внедрений», dogfooding, сертификаты, слот под логотипы (включим по согласию).

## Открытые вопросы к заказчику
- Кейсы: 3 главных клиента, измеримые результаты, что публично / под NDA.
- Apple и др. референсы → акцент + шрифт.
- EN: обе воронки или только Coordo; реальная аудитория или имидж.
- SEO-доступы: Я.Метрика / Я.Вебмастер / Wordstat + текущие позиции.
- Продукт CMS (ADR-0006 — отложено).
