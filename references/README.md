## behance-redesign-concept/
Кейс-презентация редизайна whatdadev.ru (автор: Сара, КЗ; behance.net/gallery/235170093). 11 экранов.
**Решение: НЕ берём за основу** — тёмный + красный, сервисный, без продукта (ADR-0002, 0003).
Что забираем: крупная типографика, уверенный тёмный hero, степпер процесса.
